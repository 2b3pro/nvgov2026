# Nevada Governor 2026 Voter Guide

A nonpartisan voter information site for the 2026 Nevada gubernatorial election. Compare all declared candidates on issues, records, funding, and contradictions.

**Live site:** [nvgov2026.org](https://nvgov2026.org)

## Candidates Covered

### Republican Primary
- **Joe Lombardo** (Incumbent Governor)
- **Irina Hansen** (Challenger)
- **Matthew Winterhawk** (Challenger)

### Democratic Primary
- **Aaron Ford** (Attorney General)
- **Alexis Hill** (Washoe County Commission Chair)

## Features

- Individual candidate profiles with sourced claims
- Side-by-side issue comparisons
- Primary election matchup analysis (GOP & Democratic)
- Deep dives on key issues: Housing, Economy, Crime, Education
- Citation system with hover tooltips linking to primary sources
- Mobile-responsive design

## Tech Stack

- [Vite](https://vitejs.dev/) - Build tool
- [Tailwind CSS v4](https://tailwindcss.com/) - Styling
- Static HTML - No framework dependencies

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Sources & Methodology

All claims are cited to primary sources including:
- Official government records and filings
- Court documents (PACER, state courts)
- FEC/state campaign finance data
- News reports from established outlets
- Candidate statements and campaign materials

See the [Sources page](https://nvgov2026.org/sources.html) for complete documentation.

## Disclaimer

This site uses AI-assisted research and writing. All AI-generated content has been reviewed for accuracy and grounded to primary sources. This is an independent project with no affiliation to any candidate or political party.
